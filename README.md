# Formas
- Implementar la interfaz _Figura_ y tres clases que implementan esta interfaz: Círculo, Triángulo y Rectángulo.
- La interfaz Figura debe definir un método area que calcula el área de una figura y otro método para calcular el perímetro. 
- Luego en la clase Program cree una lista de Figuras y calcule el área de cada una de las figuras de la lista.
